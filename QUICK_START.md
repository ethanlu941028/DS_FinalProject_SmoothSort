
## 🐛 常見問題

**Q: Canvas 為空？**
A: 確保輸入不為空，點擊「開始排序」

**Q: 樹結構顯示不完整？**
A: 刷新頁面，或檢查瀏覽器控制台（F12）

**Q: 速度太快？**
A: 向右拖動速度滑塊，調整為 1000ms 以上

**Q: 支援多少個元素？**
A: 理論上無限，建議不超過 20 個（視覺清晰度）

## 📊 算法複雜度

- **時間**：O(n log n)（同 Heapsort）
- **空間**：O(n)
- **穩定性**：不穩定
